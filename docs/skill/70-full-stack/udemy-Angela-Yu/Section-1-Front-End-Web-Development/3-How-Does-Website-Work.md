- 之前我们说过
    - DNS服务器可以查找网站的IP地址
    - 用户通过访问IP地址定位到server端

        - HTML
        - CSS
        - JavaScript



- HTML
    - 负责内容，比如文本，图片，按钮等
- CSS
    - 比如颜色，形状
- JS
    - 交互
    - 功能
