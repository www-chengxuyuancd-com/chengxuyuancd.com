**题目**：
- 有一对兔子，从出生后第3个月起每个月都生一对兔子，小兔子长到第三个月后每个月又生一对兔子，假如兔子都不死，问每个月的兔子总数为多少对？
**程序分析**：
* 第一个月 1对，第二个月1对，第三个月会有2对（第一个月的那一对兔子会生一对小兔子），第四个会有三对（会再生一对），第五个月有5对...
* 得到规律如下： 1, 1, 2, 3, 5, 8...
