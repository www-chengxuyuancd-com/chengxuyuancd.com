## 1. 自定义组件

### 1.1 组件的创建与引用

#### 创建组件

- 在项目根目录，鼠标右键，创建` components -> test` 文件夹
- 在新建的文件夹上，右键，创建 component
- 键入组件的名称之后回车，会自动生成组件对应的 4 个文件，后缀名分别为`.js`, `.json`, `.wxml`, `.wxss`
