## 为什么要使用企业域名邮箱

使用企业域名邮箱（如 `yourname@yourcompany.com`）而不是个人邮箱（如 `yourname@gmail.com` 或 `yourname@163.com`）有多个重要原因：

### 1. **专业形象**
企业域名邮箱展现了公司的专业性和正规性。客户和合作伙伴看到企业域名邮箱，会更加信任公司，认为公司是有组织和信誉的，而不是随意操作的小作坊或个体户。

### 2. **品牌一致性**
企业域名邮箱有助于增强品牌的一致性。每个员工使用的邮件地址都包含公司域名，有助于推广和强化品牌形象。客户在与公司沟通时，也能立即识别出品牌。

### 3. **安全性**
企业域名邮箱通常由公司管理，能够确保通信的安全性和隐私性。相比于使用个人邮箱，企业邮箱可以更好地防范钓鱼攻击、垃圾邮件和数据泄露。此外，企业还可以通过安全策略（如双因素认证）来加强邮箱的安全性。

### 4. **控制和管理**
使用企业域名邮箱使公司可以更好地管理员工的邮件账户。管理员可以创建、删除和管理员工邮箱账户，设置邮件转发规则，甚至在员工离职时保存或转移邮件记录。这种控制能力在确保业务连续性和信息安全方面至关重要。

### 5. **法律合规**
许多行业对数据存储、邮件记录保存等方面有法律合规要求。使用企业域名邮箱，可以帮助公司更好地遵守这些法规，因为公司可以对邮件进行审计和备份。

### 6. **便于协作**
企业邮箱通常与其他企业服务（如日历、文件共享、项目管理工具等）紧密集成，提升团队的协作效率。例如，使用 Google Workspace 或 Microsoft 365 的企业邮箱，员工可以轻松共享日历、文件和任务，提高工作效率。

### 7. **提升客户体验**
企业邮箱地址更易于识别和记忆，客户在联系公司时也会觉得更方便。特别是在客服和销售沟通中，使用企业域名邮箱能提升客户的信任感和满意度。

## 企业域名邮箱的申请方式

### 准备域名
如果已经有域名了，则可以跳过这个步骤

既然是域名邮箱，那么首先需要的是域名，这点可以去很多域名厂商购买即可。

注意域名是按年购买的，一年大概七八十块钱，以`.com`域名为佳，其次`.net`等，域名选择我会在后续再进行讲解，这里不做赘述，如果有问题可以先来咨询包子哥。

### 自建或者找一家邮箱服务商

这里我们可以选择一种方式以及找一家邮箱服务商用来接受你未来的邮件，可以自建（好处是不用担心未来服务商不行了），或者找一家比如 163，qq，甚至 gmail 等注册

这里我选择的是 163 注册一个：https://ym.163.com/

1. 首先输入手机号，获取验证码
![163邮箱服务商](https://chengxuyuancd.oss-cn-beijing.aliyuncs.com/auto_images/docs/work/set-up-business/link-to-customer/images/image-1.png)


2. 分别输入组织名称，你自己的名字，以及最重要的域名（一定要是你自己的域名哦）
![输入组织，域名等信息](https://chengxuyuancd.oss-cn-beijing.aliyuncs.com/auto_images/docs/work/set-up-business/link-to-customer/images/image-2.png)

3. 点击立即注册后，弹出的对话框我们可以视情况选择，但也可以和我一样，什么都不选
![注册成功](https://chengxuyuancd.oss-cn-beijing.aliyuncs.com/auto_images/docs/work/set-up-business/link-to-customer/images/image-3.png)

4. 接着便跳转到邮箱里面，进来之后先点击 最上方的 "管理后台"
![注册成功](https://chengxuyuancd.oss-cn-beijing.aliyuncs.com/auto_images/docs/work/set-up-business/link-to-customer/images/image-4-qiye-email-first-enter.png)

![域名验证第一步](https://chengxuyuancd.oss-cn-beijing.aliyuncs.com/auto_images/docs/work/set-up-business/link-to-customer/images/image-5-qiye-email-domain-validate-1.png)

8. 域名验证第一步是需要验证域名配置项

9. 上述步骤都完成后会让你设置域名生效时间，一般设置当下时间即可

## 总结

今天的分享就到这里了，本节介绍了如何申请配置域名邮箱，

你配置成功后，网易大概率会给你打电话问你要不要买付费产品，我选择了没有，因为企业还没几个人呢，可以先用免费的薅着，后续再增加

如果你不希望使用 163的邮箱服务器，也没关系，其他的邮箱服务器配置也基本相同，

最重要的就是验证域名所有权和验证域名配置，这些也只能通过域名解析的方式，所以不用担心啦
